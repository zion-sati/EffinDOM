# EffinDom Runtime

This repository contains the runtime lane for EffinDom:

- `@effindomv2/runtime` (`v2/browser-bridge`)
- Tier 1/Tier 2 runtime dependencies required to build it (`v2/core`, `v2/ui`, `v2/fonts`)

## License

## Build and publish

```bash
npm install
npm run typecheck
npm run build
npm run publish:local
```
